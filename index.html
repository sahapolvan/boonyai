📊 ผังครอบครัวประมวลผลผ่าน GitHub Pages
• รองรับการมีคู่ครองหลายคน (เช่น แก้ว, ตี๋)
• รองรับคู่ชีวิตเพศเดียวกัน (เช่น เจน-บุญเรือน, แก้ว-เบนซ์)
• เลื่อนลูกกลิ้งเมาส์เพื่อซูม หรือคลิกลากจัดตำแหน่งกล่องได้อิสระ
🎨 ความหมายสีกล่องและเส้น:
กล่องสีฟ้า = เพศชาย (ช)
กล่องสีส้ม = เพศหญิง (ญ)
เส้นสีแดง = คู่ชีวิต / แต่งงาน
เส้นสีน้ำเงิน + ลูกศร = ชี้ไปหาลูก
